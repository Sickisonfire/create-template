# create-template cli tool

CLI Tool for template creation

**Note: not tested on windows.**

## install

Git clone and cd into it

#### yarn

```
yarn global add $PWD
```

#### npm

```
npm i -g
```

## usage

```
create-template
```

## uninstall

#### yarn

```
yarn global remove create-template
```

#### npm

```
npm uninstall -g create-template
```
